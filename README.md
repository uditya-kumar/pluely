## 🛠️ Changes in This Fork

- **Original repo:** [iamsrikanthnani/pluely](https://github.com/iamsrikanthnani/pluely)
- **This modified version:** [uditya-kumar/pluely (v1.01 release)](https://github.com/uditya-kumar/pluely/releases/tag/v1.01)

**New Features**

- Solve shortcut (`Ctrl/Cmd+Shift+X`) — sends all captured screenshots to the AI with a configurable prompt.
- Copy Response shortcut (`Ctrl/Cmd+Shift+C`) — copies the current AI response, no window focus needed.

**Paywall Removals (all premium gates unlocked)**

- Drag window button no longer requires a license.
- Shortcut customization (rebinding key bindings) unlocked.
- Screenshot Selection Mode unlocked.
- Response Settings (length, language, auto-scroll) unlocked.
- Theme customization and window transparency unlocked.
- Chat follow-up message input unlocked.
- Move Window shortcut (`Ctrl/Cmd + Arrow keys`) unlocked.

**Bug Fixes**

- App Icon Stealth Mode now actually hides the taskbar icon on Windows.
- Toggle Window (`Ctrl/Cmd+\`) no longer steals keyboard focus.
- Response panel no longer stays hidden/flickers after a hide→show cycle.
- Fixed "HotKey already registered" errors when toggling/rebinding shortcuts.
- Closing the main window now fully quits the app instead of running in the background.

> ℹ️ Paywall removals affect only your local build — respect upstream licensing if redistributing.
